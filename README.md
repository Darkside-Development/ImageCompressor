# Image Compressor
Compress an image with this simple script made in Python.
# Usage
`git clone https://github.com/Darkside-Development/ImageCompressor`<br>

`cd ImageCompressor-Main`<br>

Drag your picture into the folder.<br>

`python3 main.py`
